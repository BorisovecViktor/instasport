## How  to run
 1. Clone the repository `git clone https://github.com/BorisovecViktor/react-redux-typescript.git`
 2. Go into project folder `cd react-redux-typescript`
 3. Run `npm install`
 4. Run `npm start` 
 5. Open in your Browser `http://localhost:3000`
